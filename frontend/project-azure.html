MICROSOFT AZURE HACKING LAB

Hacking Lab

Built a hacking lab in Microsoft Azure to simulate an enterprise network and vulnerable machines. Deployed a domain controller on Windows Server 2022 along with two Windows Server 2019 machines utilizing Azure Virtual Machines. Deployed two Windows 11 VMs and joined to the domain. Created multiple VNETS within the Resource Group to segment the network.

This lab is used for penetration testing practice, vulnerability assessment, and learning Active Directory security.
